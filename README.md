# SpinTyre Web

Marketing site for **SpinTyre** — the short-form car video app for iPhone, Android, and Apple Watch.

Live: [https://spintyre.netlify.app](https://spintyre.netlify.app)

## Pages

- `/` — product landing page
- `/privacy-policy` — privacy policy
- `/delete-account` — account deletion instructions
- `/data-safety` — Google Play / App Store data safety summary

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Deployed on Netlify from this repo.
